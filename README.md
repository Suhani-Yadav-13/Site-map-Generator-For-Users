# Sitemap Generator for Users
# Project Overview - 
Sitemap Generator is an offline utility that generates XML sitemaps from local HTML files without requiring a live website or internet connection. 
It scans local folders, extracts internal links, and builds a structured sitemap that can be previewed in a visual tree view and downloaded for later use.
This tool is especially useful during static website development and SEO preparation before deployment.

# Key Features - 
* Generate sitemap from local HTML files (offline)
* Automatic detection of internal links
* Visual tree view to understand site structure
* Downloadable XML sitemap
* No server or internet connection required
* Simple and user-friendly interface

# Tech Stack - 
* HTML
* CSS
* JavaScript
* EJS

# How It Works - 
* User selects a local folder containing HTML files
* The tool scans files and extracts internal links
* A hierarchical sitemap structure is generated
* Users can:
            View the sitemap in tree format
            Download the sitemap as an HTML file

# How to Run the Project - 
1. Clone the repository:
   git clone https://github.com/Suhani-Yadav-13/Site-map-Generator-For-Users.git
2. Open the project folder
3. Go inside the backend folder
4. Write "npm start" Command
5. Select your local HTML project folder to generate the sitemap
